# User Stories

## Site Administrator

- As a **site admin**, I want to **add a custom theme** so that **courses look personalised for the college/university**.

## Teacher

- As a **teacher**, I want to **add coding questions** so that **students can attempt them online**.
- As a **teacher**, I want to **have automatic grading of quizzes** so that **I can save time and focus on teaching**.
- As a **teacher**, I want to **have access to a question bank** so that **I can easily add questions**.

## Student

- As a **student**, I want to **have a text editor in the question** so that **I can attempt questions from inside the course**.
- As a **student**, I want to **submit my submit my solution and see output** so that **I can check the correctness of my solution**.
- As a **student**, I want to **see a readability report** so that **I can write more readable code**.
